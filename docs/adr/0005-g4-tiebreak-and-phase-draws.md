# ADR 0005 — Desempate do G4 e mata-mata fase a fase

**Status:** Aceito  
**Data:** 2026-07-11

## Contexto

Quatro dos cinco participantes de cada grupo avançam. Os critérios esportivos podem não separar todos os concorrentes, e o campeonato não possui chaveamento predeterminado entre posições dos grupos.

## Decisão

- Empate absoluto só gera partidas extras quando atravessa o corte entre quarto e quinto.
- Com dois envolvidos, ocorre uma partida decisiva com pênaltis em caso de empate.
- Com três ou mais, ocorre turno único; se o corte continuar indefinido, o processo se repete apenas entre os ainda empatados.
- Empates absolutos que não afetam o corte compartilham posição e usam ordem alfabética somente para apresentação.
- Os oito classificados são sorteados livremente nas quartas, inclusive contra adversários do mesmo grupo.
- Os quatro vencedores são sorteados novamente nas semifinais.
- A final é criada automaticamente com os dois vencedores das semifinais.
- Não existem oitavas ou disputa de terceiro lugar.

## Consequências

### Positivas

- A vaga do G4 nunca é decidida por critério administrativo arbitrário.
- A imprevisibilidade do mata-mata é preservada em cada fase.
- Posições internas do G4 não oferecem vantagem indevida.

### Negativas

- Empates múltiplos podem prolongar a fase de grupos.
- É necessário persistir séries de desempate separadas das estatísticas regulares.
- A chave não pode pré-renderizar conexões reais de fases ainda não sorteadas.
